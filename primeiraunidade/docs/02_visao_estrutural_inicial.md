# Visao estrutural inicial

A visao estrutural segue o mesmo padrao do projeto de referencia:

- pacote `SysADL.types`;
- pacote `SysADL_ports`;
- pacote `SysADL_connectors`;
- pacote `SysADL_components`.

O BDD deve mostrar os elementos definidos nesses pacotes. O IBD deve mostrar a configuracao inicial de `SistemaHotelResortCP`.

Esta versao e propositalmente inicial. A 3a avaliacao devera revisar e refinar essa estrutura e acrescentar a visao comportamental.
