# Projeto Hotel/Resort - 1a avaliacao

Projeto SysADL inicial criado seguindo o padrao do projeto de referencia `ProjetoArq`.

## Importante

Esta versao foi feita para abrir no Eclipse/SysADL com a mesma estrutura do projeto de referencia:

- `My.sysadl` na raiz;
- `.project`;
- `.settings/org.eclipse.core.resources.prefs`;
- `representations.aird` com as representacoes de requisitos, BDDs e IBD.

O conteudo foi adaptado para o tema de hotel/resort e para o modelo de requisitos do projeto, sem copiar a proposta do projeto de referencia.

## Escopo da 1a avaliacao

Inclui:

- modelo de requisitos;
- tipos iniciais;
- ports;
- connectors;
- components;
- configuracao estrutural inicial.

Nao inclui ainda a visao comportamental completa da 3a avaliacao.
